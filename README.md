# php-helpers

A lightweight and reusable collection of common PHP helper functions.

## ✨ Features

- Framework-agnostic
- No dependencies
- Clean and consistent API
- PHP 7.4+ compatible

## 🛠 Requirements

- PHP 7.4 or higher

## 📦 Installation

Install via Composer:

```bash
composer require dragontnt/php-helpers
```
